🧠 About This Project

The mission, the process, the tech — and the student behind the madness.

👨‍💻 Why I Built This

This project wasn’t built to tick boxes. It was built to prove that learning can be visual, interactive, and full of power. I wanted to combine design, code, science, and creativity into one unforgettable experience. And I did it all by hand.

🚀 What It Includes

This is a full-stack front-end experience. You’ll find:

  • A cinematic homepage with parallax planets and a glowing theme
  • A 2D solar system animation crafted with CSS only
  • A 3D interactive spinning sphere made with Three.js
  • Real content written for real learning — not just placeholders

🧰 Tools I Used

Every part of this website was handcrafted using the following technologies:

  • HTML5: Structure, layout, page content
  • CSS3: Styling, animation, hover effects, themes
  • VS Code: My coding battlefield

🔥 Challenges I Faced

Making real animations by code means breaking things — a lot. I struggled with transform math, orbit formulas, and CSS rendering bugs. But every mistake taught me how this tech really works. That’s how you learn.

📈 What I Learned

  • How to animate with CSS and keyframes
  • How to control a 3D scene with camera + lighting
  • How to write clean, readable, professional-grade code
  • How to design for usability and readability

💡 Final Thoughts

This project proves what a single student can build with nothing but imagination and code. It was devil-raged from day one — and I’m proud to say: this website will not be forgotten.
